# 📚 Liane's Library Advanced SQL, Python and Streamlit App

This project demonstrates a full SQL-to-application workflow using a library management case study.

The project includes database design, advanced SQL querying, connecting a SQL database with Python, and building a Streamlit app to interact with the library data.

## Project Overview

Liane's Library is a small library management case study focused on books, friends, and borrowing records.

The goal of the project is to design a relational database, write SQL queries to manage and analyse the data, connect the database with Python, and build a simple Streamlit interface for practical use.

## Workflow

```text
SQL database design
        ↓
Table creation and sample data
        ↓
Advanced SQL queries
        ↓
Python database connection
        ↓
Streamlit application
```

## Key Areas Covered

- Relational database design

## Project Components

### SQL

The SQL part of the project contains the database structure, table creation scripts, sample data, and advanced queries used to manage and analyse the library data.

### Python

The Python part focuses on connecting to the SQL database, executing queries, and working with query results in Pandas.

### Streamlit App

The Streamlit app provides a simple interface for interacting with the library data, such as viewing books, checking availability, and managing borrowing records.
